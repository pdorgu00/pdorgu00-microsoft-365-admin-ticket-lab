# Lab 01 – User Lifecycle Administration

## Overview

This lab demonstrates the complete Microsoft 365 user lifecycle by creating a new user account, managing account settings, resetting passwords, blocking sign-in, and reviewing licensing options.

---

## Scenario

A new employee joins the organization and requires a Microsoft 365 account. After the account is created, several common administrative tasks are performed to simulate a real-world help desk ticket.

---

## Tasks Completed

- Created a new Microsoft 365 user
- Configured user profile information
- Verified successful account creation
- Reviewed Active Users
- Reset the user's password
- Blocked user sign-in
- Verified sign-in restrictions
- Reviewed Microsoft 365 license assignment

---

## Skills Demonstrated

- Microsoft 365 Admin Center
- User Lifecycle Management
- Identity and Access Management (IAM)
- Password Administration
- Account Security
- License Management
- Help Desk Administration
