# Microsoft 365 Administration Lab

## Overview

This repository documents hands-on Microsoft 365 administration tasks completed in a Microsoft 365 test tenant. The project demonstrates common help desk and junior administrator responsibilities including user account management, Microsoft Teams administration, Microsoft 365 Groups, and Exchange Online shared mailbox configuration.

The lab environment was built to simulate real-world administrative tasks commonly performed in enterprise environments.

---

## Environment

- Microsoft 365 Admin Center
- Microsoft Teams Admin Center
- Exchange Admin Center
- Microsoft 365 Test Tenant

---

## Lab Contents

### Lab 01 – User Lifecycle Administration

- Create a Microsoft 365 user
- Configure user information
- Verify user creation
- Reset user password
- Block user sign-in
- Review license assignment

### Lab 02 – Teams and Group Administration

- Navigate Microsoft Teams Admin Center
- Review Teams dashboard
- Manage Microsoft 365 Groups
- Assign group owners
- Add group members
- Verify membership

### Lab 03 – Shared Mailbox Configuration

- Create a shared mailbox
- Review mailbox properties
- Assign mailbox permissions
- Configure automatic replies
- Verify mailbox configuration
