# Lab 02 – Teams and Group Administration

## Overview

This lab demonstrates Microsoft Teams and Microsoft 365 Group administration in a Microsoft 365 test tenant. Administrative tasks include navigating the Teams Admin Center, managing Microsoft 365 groups, assigning owners, adding members, and verifying group membership.

---

## Scenario

An organization requires a Microsoft Teams environment for collaboration. A Microsoft 365 group is reviewed and managed by assigning owners, adding members, and verifying the group's configuration.

---

## Tasks Completed

- Accessed the Microsoft Teams Admin Center
- Reviewed the Teams dashboard
- Navigated the Users section
- Reviewed Microsoft 365 Groups
- Opened group details
- Assigned group owners
- Added group members
- Verified group membership

---

## Skills Demonstrated

- Microsoft Teams Administration
- Microsoft 365 Groups
- Group Ownership Management
- Membership Administration
- Collaboration Administration
- Microsoft 365 Admin Center
- Help Desk Documentation
