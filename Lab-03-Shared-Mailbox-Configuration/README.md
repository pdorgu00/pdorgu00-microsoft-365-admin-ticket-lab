# Lab 03 – Shared Mailbox Configuration

## Overview

This lab demonstrates Exchange Online shared mailbox administration within a Microsoft 365 test tenant. Administrative tasks include creating a shared mailbox, reviewing mailbox properties, assigning permissions, and configuring automatic replies.

---

## Scenario

A shared mailbox is created for an organization's support team. Administrative tasks include creating the mailbox, verifying successful creation, reviewing mailbox settings, and configuring automatic replies for incoming email.

---

## Tasks Completed

- Accessed Exchange Online shared mailboxes
- Created a shared mailbox
- Verified successful mailbox creation
- Reviewed mailbox properties
- Configured automatic replies
- Verified mailbox configuration

---
